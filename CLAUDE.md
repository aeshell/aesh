## Knowledge Tree (MANDATORY)

A persistent knowledge tree is maintained at ~/git/llm_knowledge/.
Full instructions are in ~/.claude/claude.md — read that file at session start.

The project name is the root directory name of this repo in lowercase (e.g., if the repo is at ~/git/Foo, the project is "foo").

**Before starting ANY work, read these files (blocking prerequisite):**
1. ~/git/llm_knowledge/<project>/rules.md
2. ~/git/llm_knowledge/<project>/hypotheses.md
3. ~/git/llm_knowledge/<project>/knowledge.md
4. ~/git/llm_knowledge/_shared/ — read rules.md in each topic folder

**After completing a task (before reporting done):**
Update ~/git/llm_knowledge/<project>/ with non-obvious insights.
If an insight applies across projects, add it to ~/git/llm_knowledge/_shared/<topic>/.
Create new _shared/ topics as needed. Update index.md if new folders are added.
Do NOT wait to be asked — update proactively when the work revealed something surprising.
